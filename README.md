# sidneygonzalez.ai

Data-Driven Decision Intelligence Portfolio — a single-page React application demonstrating executive-grade financial intelligence systems with live Python computation.

## What This Is

Three interconnected projects form a unified Corporate Intelligence Stack:

1. **Executive Financial Command Center** — The dashboard a CFO would actually use
2. **Cross-Functional Decision Impact Analyzer** — The what-if engine for C-suite decisions
3. **Automated Variance & Anomaly Reporting** — The early warning system

Each project demonstrates the full pipeline: messy data in, clean insight out — with live Python (Pyodide) running in the browser.

## Tech Stack

| Layer | Technology |
|-------|-----------|
| Framework | React 18 (Vite) |
| Styling | Tailwind CSS 3.x |
| Animation | Motion v12 + Lenis smooth scroll |
| Charts | Recharts + Nivo |
| Math | KaTeX |
| Python Runtime | Pyodide (Web Worker) |
| Deployment | Cloudflare Pages |
